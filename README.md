dingcan
=======

##点餐系统
参考 餐库：https://github.com/willerce/canku


_基于餐库上做了修改，使用了php进行重写_
_结合RTX接口认证，直接使用内部RTX账号密码进行登陆_


=======
##功能

_餐类分为午餐、晚餐，以14点为临界点_
_购物车功能_
_订单功能_
_历史消费记录查看_


=======
##后台功能

_添加、删除、修改店铺，店铺显示等_
_添加、删除、修改美食分类和美食_
_用户订单管理：取消、帮忙支付等_
_设置公司补贴金额，超出部分才扣费_
_一键RTX通知用户点餐_
