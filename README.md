dingcan
=======

##点餐系统
参考 餐库：https://github.com/willerce/canku


1、基于餐库上做了修改，使用了php进行重写
2、结合RTX接口认证，直接使用内部RTX账号密码进行登陆


=======
##功能

1、餐类分为午餐、晚餐，以14点为临界点
2、购物车功能
3、订单功能
4、历史消费记录查看


=======
##后台功能

1、添加、删除、修改店铺，店铺显示等
2、添加、删除、修改美食分类和美食
3、用户订单管理：取消、帮忙支付等
4、设置公司补贴金额，超出部分才扣费
5、一键RTX通知用户点餐
